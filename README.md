# blog_foxs

Dự án sử dụng
1/ API-Cosmicjs: https://cosmicjs.com/
2/ Front-end: - Vue: https://vuejs.org/ - Vuetify: https://vuetifyjs.com/en/ - Vuex: https://vuex.vuejs.org/
